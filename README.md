# Quiz App

Recap Project Quiz App

## Running the project

This project is vanilla HTML, CSS and Javascript. To run it, just open the one of the HTML pages

## Overview

This project is a mobile-friendly quiz application. It allows users to read questions, bookmark them, and view their profile. The app has three main pages:

- **Home Page:** Displays all available questions for users to browse.
- **Bookmarks Page:** Lists all questions that the user has bookmarked for easy reference.
- **Profile Page:** Allows users to view and manage their profile.

The layout of the app is built to be responsive, with a focus on clean, structured code using the BEM methodology for CSS.

## Feature

- Read quiz questions
- Bookmark questions for later review
- Navigate through different pages: Home, Bookmarks, Profile
- Responsive layout for mobile and web platforms
